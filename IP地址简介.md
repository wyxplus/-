# IP 地址



## 1. 私有 IP 

 A 类地址： 10.0.0.0/8  （10.0.0.0 ~ 10.255.255.255）

 B 类地址： 172.16.0.0/12 （172.16.0.0 ~ 172.31.255.255）

 C 类地址： 192.168.0.0/16  （192.168. 0.0 ~ 192.168.255.255）



## 2. 特殊 IP

|    全0    |      本主机      |
| :-------: | :--------------: |
| 全0\|主机 | 本网络上的本主机 |
|    全1    |   本网络中广播   |
| 网络\|全1 |   网络定向广播   |
| 127\|任意 |     环回地址     |

## 4. VLSM （变长子网掩码）

可以对一个子网再进行子网划分。

优点：

对 IP 地址更为有效的使用

应用路由归纳的能力更强



## 5. IP 协议

### 1. 不可靠：

​	它不能保证 IP 数据报能成功到达目的地址。

可靠性由上层协议保证。



### 2. 无连接：

​	不维护任何关于后续数据报的状态信息，有可能乱序。
